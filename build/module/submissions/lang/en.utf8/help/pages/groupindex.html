Submissions within a group

The table contains the portfolios that have been submitted for assesment to this group. Only group administrators (and tutors if this is a "Course" group) have access to this page.

Group administrators can assign individual assessors to portfolios to split up the responsibilities among a group of assessors.

Use the "Search" field to search for anything relating to the submission, e.g. the name of the portfolio author, the title of the portfolio, or the name of the assessor.

Use the "Quick filters" to narrow down the portfolios shown in the table. These quick filters are "sticky", i.e. they stay set until you change them:

  • Tasks associated with submissions
  • Assessor assigned to the submissions
  • Feedback placed
  • Result of the assessment
  • Status of the submitted portfolio

You can decide which columns to display in the table by using the "Configure columns" button.

Sort the table according to the column header that you prefer. Each column can be sorted in ascending and descending order.
