Files

This is your files area where you can upload your documents either for display purposes or for safe-keeping. You initially have 200MB of space, but this can be increased to 1GB on request (Contact ltu@solent.ac.uk).

You have the choice of uploading a file or creating a folder to upload a file into. Folders can help keep this area tidy and is recommended. When uploading, tick the 'Upload File' box where a 'browse' option will become available.
